# # Cardápio Virtual - Páscoa 2026 🐰🍫

Este é um cardápio virtual elegante e moderno desenvolvido sob medida para a **Confeitaria Artesanal Camila Santos**. O projeto foi desenhado no Figma com a premissa de um web design fluido, com ricas animações suaves e alta legibilidade, ideal para apresentar os produtos da campanha de Páscoa 2026.

## 🌟 Tecnologias Utilizadas
O site foi construído para ser o mais rápido e leve possível, totalmente sem dependências e frameworks externos (como React ou Bootstrap), utilizando apenas:
- **HTML5**: Semântico e acessível.
- **CSS3 Vanilla**: Estilização com Flexbox/CSS Grid, CSS Variables e media queries para responsividade extrema. Efeitos modernos usando `backdrop-filter` e micro-animações como `hover` em imagens.
- **JavaScript Moderno (ES6)**: Utilizado estritamente para a injeção do catálogo de preços de forma escalável.

## 🎨 Design e Tipografia
- **As cores base do projeto:**
  - Azul Noturno (`#0A1E4C`) e tons derivados
  - Textos de escopo legível (`#6C7289` e branco neve)
- **Tipografia:** 
  - A fonte principal, importada localmente, é a majestosa **Thenie Dawson**, adicionando um peso visual fino e atrativo;
  - A fonte secundária (para as descrições dos doces) é a moderna **Poppins**, servida via Google Fonts.

## ✨ Destaques
- **Sidebar Animada**: A barra lateral esquerda (no Desktop) contém a logo redonda oficial, contatos com o botão de autoatendimento WhatsApp e fica fixa para facilitar a navegação.
- **Grade Automática (Grid View)**: As fotos e descrições dos doces (Ovos de Colher, Kit Páscoa, Surpresa de Uva, etc) se adaptam sozinhas à largura da tela.
- **100% Responsivo**: No celular, a *sidebar* se posiciona como um hero cabaçalho e as margens são redistribuídas para leitura touch-friendly. As imagens flutuam graciosamente a cada passada e seleção de item.
